# 🧴 AcneAI Analyzer

A comprehensive web application for acne recognition, analysis, and personalized treatment recommendations using AI and computer vision.

## 🚀 Features

- **📸 Camera-based Analysis**: Real-time skin analysis using device camera
- **🤖 AI-Powered Detection**: Identify different acne types and severity
- **📋 Interactive Questionnaire**: Personalized assessment based on lifestyle
- **💡 Treatment Recommendations**: Customized skincare routines
- **📊 Progress Tracking**: Monitor skin improvement over time

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript, TensorFlow.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (planned)
- **AI/ML**: Python, TensorFlow (planned)

## 🚀 Quick Start

1. **Clone the repository**
   \`\`\`bash
   git clone https://github.com/Neeha-shetty/ACNEANALYSIS
   cd acne-ai-website
   \`\`\`

2. **Start the backend server**
   \`\`\`bash
   cd backend
   npm install
   node server.js
   \`\`\`

3. **Open your browser**
   - Main app: \`http://localhost:5001/acne-camera-working.html\`
   - Homepage: \`http://localhost:5001/index.html\`

## 🎯 Usage

1. Allow camera access when prompted
2. Take a clear photo of your skin
3. Complete the lifestyle questionnaire
4. Receive personalized analysis and recommendations
5. Track your progress over time
